# @mayrlabs/core

Core shared library and CLI for MayR Labs.

## Installation

```bash
npm install @mayrlabs/core
```

## CLI Usage

```bash
npx @mayrlabs/core
# or
npx mayrlabs
```

## Features

- **Interactive CLI**: Access Github, Website, and Packages.
- **Shared Utilities**: Provides standard `introScreen` and brand constants.
